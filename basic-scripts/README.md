# Basic Scripts

Original Selenium automation scripts - learning and experimentation files.

## Files

- `brave.py` - Basic Brave browser automation
- `chrome.py` - Chrome browser setup
- `choose.py` - Browser selection script
- `search.py` - Simple search automation
- `logs.txt` - Execution logs

## Usage

These are basic scripts without test frameworks. Run them directly:

```bash
python brave.py
python search.py
```

## Note

For production-ready tests with proper structure, see:
- `../python-tests/` - Python tests with pytest
- `../java-tests/` - Java tests with TestNG
